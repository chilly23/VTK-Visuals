# AgentGuard v7 — Setup & Deploy

## Load in Chrome (2 min)
1. chrome://extensions/ -> Enable Developer Mode
2. Load unpacked -> select the agentguard_v7/ folder
3. Done

## Use
1. Open a normal Chrome tab (YouTube, Google, Amazon — any page)
2. Click the AgentGuard toolbar icon
3. Sidebar opens. Type your task. Press RUN AGENT or Ctrl+Enter.

The agent runs entirely in the sidebar window — it will open/control your browser
tab in front of you. You keep watching the sidebar for live status.

## Examples
  Play Interstellar trailer on YouTube
  Search flights from Bangalore to Delhi on Ixigo
  Find OnePlus 12 price on Amazon
  Latest AI news on Google News

## After task completes — follow up
Type a follow-up task. The agent remembers the full conversation.
  > "now click the second video"
  > "filter by price: low to high"
